# Приложение списка товаров

React-приложение для управления каталогом товаров с авторизацией, поиском, сортировкой и пагинацией.

## 🚀 Быстрый старт

```bash
# 1. Скопировать файл с переменными окружения
cp env.sample .env

# 2. Установить зависимости
npm install

# 3. Запустить приложение в режиме разработки
npm run dev

```

Приложение будет доступно по адресу: http://localhost:5173
